评估指标说明

准确率 (Accuracy)

正确分类的样本占总样本的比例

精确率 (Precision)

预测为正类的样本中真正为正类的比例

召回率 (Recall)

所有正类样本中被正确预测的比例

F1值 (F1-Score)

精确率和召回率的调和平均值
