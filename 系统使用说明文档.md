# 医疗文献智能检索系统 - 使用说明文档

## 一、系统简介

### 1.1 系统概述
医疗文献智能检索系统是一个基于自然语言处理技术的医学文献检索与分析平台。系统集成了关键词检索、语义检索、文献自动分类、相似文献推荐和自动摘要等功能，旨在帮助医学研究人员快速准确地查找和分析相关文献。

### 1.2 主要功能
- **双模式检索**：关键词检索 + 语义检索
- **智能分类**：7大医学主题自动分类
- **相似推荐**：基于向量相似度的文献推荐
- **自动摘要**：文献核心内容智能提取
- **数据管理**：文献导入、导出、批量处理
- **模型训练**：在线训练和更新分类模型

### 1.3 技术特点
- 基于BERT预训练模型的中文文本理解
- 使用Django框架的Web应用架构
- RESTful API设计，支持前后端分离
- 响应式界面设计，适配多终端设备

## 二、系统安装与部署

### 2.1 环境要求

#### 2.1.1 硬件要求
- **CPU**：4核及以上（推荐8核）
- **内存**：8GB及以上（推荐16GB）
- **硬盘**：20GB可用空间
- **GPU**：可选（支持CUDA的NVIDIA显卡可加速推理）

#### 2.1.2 软件要求
- **操作系统**：Ubuntu 20.04+/CentOS 7+/Windows 10+
- **Python**：3.8或更高版本
- **数据库**：SQLite（开发）/ PostgreSQL 12+（生产）
- **Web服务器**：Nginx（生产环境推荐）

### 2.2 安装步骤

#### 2.2.1 基础环境准备
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install python3 python3-pip python3-venv git

# CentOS/RHEL
sudo yum install python3 python3-pip python3-virtualenv git

# Windows
# 1. 安装Python 3.8+ from python.org
# 2. 安装Git from git-scm.com
```

#### 2.2.2 克隆代码仓库
```bash
git clone https://github.com/yourusername/Intelligent-Medical-Literature-Retrieval.git
cd Intelligent-Medical-Literature-Retrieval
```

#### 2.2.3 创建虚拟环境
```bash
# Linux/Mac
python3 -m venv venv
source venv/bin/activate

# Windows
python -m venv venv
venv\Scripts\activate
```

#### 2.2.4 安装系统依赖
```bash
# 安装系统核心依赖
cd system_code
pip install -r requirements.txt

# 安装BERT模型依赖
cd ../BERT
pip install -r requirements.txt
```

#### 2.2.5 数据库初始化
```bash
# 返回系统目录
cd ../system_code

# 创建数据库迁移
python manage.py makemigrations
python manage.py migrate

# 创建超级用户（用于管理后台）
python manage.py createsuperuser
```

#### 2.2.6 静态文件收集
```bash
python manage.py collectstatic
```

#### 2.2.7 数据导入
```bash
# 导入初始分类数据
python manage.py shell
# 在Python shell中执行：
from literature.models import LiteratureCategory
categories = ['肿瘤学', '内分泌代谢', '心血管', '神经内科', '感染性疾病', '中医药', '其他']
for cat in categories:
    LiteratureCategory.objects.get_or_create(name=cat)
exit()

# 导入文献数据（可选）
python import_data.py
```

#### 2.2.8 启动开发服务器
```bash
python manage.py runserver 0.0.0.0:8000
```

### 2.3 生产环境部署

#### 2.3.1 使用Gunicorn
```bash
# 安装Gunicorn
pip install gunicorn

# 启动Gunicorn（在system_code目录下）
gunicorn medical_search.wsgi:application -w 4 -b 127.0.0.1:8000
```

#### 2.3.2 Nginx配置
```nginx
server {
    listen 80;
    server_name your_domain.com;
    
    location /static/ {
        alias /path/to/Intelligent-Medical-Literature-Retrieval/system_code/static/;
    }
    
    location /media/ {
        alias /path/to/Intelligent-Medical-Literature-Retrieval/system_code/media/;
    }
    
    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
}
```

#### 2.3.3 进程管理（Supervisor）
```ini
[program:medical_search]
command=/path/to/venv/bin/gunicorn medical_search.wsgi:application -w 4 -b 127.0.0.1:8000
directory=/path/to/Intelligent-Medical-Literature-Retrieval/system_code
user=www-data
autostart=true
autorestart=true
redirect_stderr=true
```

### 2.4 Docker部署（可选）
```dockerfile
# Dockerfile
FROM python:3.9-slim

WORKDIR /app

COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY . .

RUN python manage.py collectstatic --noinput

EXPOSE 8000

CMD ["gunicorn", "medical_search.wsgi:application", "--bind", "0.0.0.0:8000"]
```

## 三、用户操作指南

### 3.1 系统首页

#### 3.1.1 访问系统
在浏览器中访问：`http://localhost:8000` 或您的服务器地址

#### 3.1.2 首页功能
1. **搜索框**：输入关键词或问题，选择检索类型
2. **热门分类**：点击分类标签快速查看相关文献
3. **系统功能**：查看系统六大功能介绍
4. **数据统计**：查看文献总数、分类数量等统计信息
5. **最新文献**：浏览系统中最新的文献列表

### 3.2 文献检索

#### 3.2.1 关键词检索
1. 在首页搜索框中选择"关键词检索"
2. 输入检索词（如"糖尿病治疗"）
3. 点击"检索"按钮或按Enter键
4. 查看检索结果列表

**检索特点**：
- 支持标题、摘要、关键词的精确匹配
- 自动分词，提高检索召回率
- 支持分页浏览

#### 3.2.2 语义检索
1. 在首页搜索框中选择"语义检索"
2. 输入自然语言问题（如"关于糖尿病的最新治疗方法"）
3. 点击"检索"按钮
4. 查看按相似度排序的结果

**检索特点**：
- 基于BERT模型理解查询意图
- 返回语义相关的文献
- 显示相似度分数

#### 3.2.3 高级检索技巧
- **多关键词**：用空格分隔多个关键词
- **精确短语**：使用引号包围短语（如"靶向治疗"）
- **分类筛选**：在结果页面使用分类筛选器
- **时间排序**：支持按时间排序结果

### 3.3 文献浏览

#### 3.3.1 文献列表页
访问路径：点击导航栏"文献列表"或首页"查看更多文献"

**功能**：
- 按分类筛选文献
- 关键词搜索文献
- 分页浏览所有文献
- 查看文献基本信息

#### 3.3.2 文献详情页
访问路径：点击文献列表中的文献标题

**显示内容**：
- 文献完整标题
- 作者、期刊、发表年份
- 完整摘要
- 关键词
- 所属分类
- 相似文献推荐

**操作**：
- 查看相似文献
- 生成自动摘要
- 下载PDF（如有）

### 3.4 分类功能

#### 3.4.1 单文本分类
访问路径：导航栏"分类功能" → "单文本分类"

**操作步骤**：
1. 在文本框中输入文献标题和摘要
2. 点击"分类"按钮
3. 查看分类结果和置信度

#### 3.4.2 批量自动分类
访问路径：导航栏"分类功能" → "批量分类"

**操作步骤**：
1. 点击"开始批量分类"按钮

### 3.5 模型训练

#### 3.5.1 训练页面
访问路径：导航栏"模型训练"

**功能**：
- 查看当前模型信息
- 配置训练参数

#### 3.6.1 数据导入

**导入方式**：

## 五、常见问题解答

### 5.1 安装问题

#### Q1: 安装依赖时遇到权限错误
**A**: 使用虚拟环境或添加`--user`参数：
```bash
pip install --user -r requirements.txt
```

#### Q2: 数据库迁移失败
**A**: 删除数据库文件重新迁移：
```bash
rm db.sqlite3
python manage.py migrate
```

#### Q3: 导入数据时编码错误
**A**: 确保使用正确的编码：
```bash
# 在import_data.py中添加编码参数
df = pd.read_csv(file_path, encoding='utf-8-sig')
```

### 5.2 使用问题

#### Q1: 搜索无结果
**A**：
1. 确认数据库中有文献数据
2. 尝试更简单的关键词
3. 检查是否选择了正确的检索类型

#### Q2: 分类结果不准确
**A**：
1. 重新训练模型
2. 确保输入文本包含足够信息
3. 调整分类阈值

#### Q3: 系统响应慢
**A**：
1. 首次使用需要加载BERT模型
2. 大量数据检索时使用分页
3. 生产环境使用缓存和CDN

### 5.3 模型训练问题

#### Q1: 训练时内存不足
**A**：
1. 减小批处理大小
2. 缩短文本最大长度
3. 使用GPU加速

#### Q2: 训练准确率低
**A**：
1. 增加训练数据量
2. 调整学习率
3. 增加训练轮数
